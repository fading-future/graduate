绿色的是数据

红色的是命令

橙色和黄色的是展示



`Ortho Slice`  橙色；展示三维数据的一个切片信息

`Bounding Box`  黄色；使用一个框框定数据范围，并展示出来

`Volume Rendering` 黄色；展示三维数据

`Volume Edit` 红色；选择展示/渲染范围

`Interactive Thresholding` 红色；做交互式阈值分割
